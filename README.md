# Syst-me-d-information-HomeAppShop
HomeAppShop modélise les exigences d'un SI omnicanal d'électroménager (vente, stock, logistique, finance) développé à l'AGH de Cracovie. Ce projet couvre le cycle SDLC et démontre une démarche rigoureuse de Business Analyst en traduisant les besoins métiers en spécifications fonctionnelles (UML, Cockburn) prêtes pour le développement
